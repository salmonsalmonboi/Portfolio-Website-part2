# Portfolio Website

## Task2: part2
เว็บไซต์ Portfolio นี้จัดทำขึ้นเพื่อแสดงผลงานและทักษะของเราในด้านการพัฒนาเว็บไซต์ โดยเน้นประสบการณ์ผู้ใช้ (UI/UX) และการสร้างแอปพลิเคชันที่ตอบสนองได้ดี   
**Live Site**: [supakornsiriportfolio.netlify.app](https://portfolio-website-part2.netlify.app/)  
**เอกสารอธิบายงาน**: [Google Docs](https://docs.google.com/document/d/1rm6WZ6TUCqlJI2FUzBIRRLrszSD1VmEJhVqEkLExGWg/edit?usp=sharing)


## Features ที่เพิ่มเติม

- 🌙 **Dark Mode Toggle** – เปลี่ยนธีมเว็บไซต์ระหว่างโหมดสว่างและมืด เพื่อให้เหมาะกับความสบายตาของผู้ใช้
- ⏳ **Loading Animation** – หน้าจอโหลดแบบมืออาชีพ ช่วยเพิ่มความประทับใจขณะรอเว็บไซต์แสดงผล
- 📊 **Scroll Progress Indicator** – แสดงแถบความคืบหน้าการเลื่อนหน้า เพื่อให้ผู้ใช้รู้ตำแหน่งเนื้อหาที่กำลังอ่าน
- 🗂 **Project Filter System** – ระบบกรองโปรเจกต์ตามหมวดหมู่ ช่วยให้ผู้เข้าชมค้นหาผลงานได้ง่ายขึ้น
- ⌨️ **Typing Animation** – เอฟเฟกต์พิมพ์ข้อความในส่วน Hero เพิ่มความเคลื่อนไหวและความน่าสนใจให้หน้าแรก
- ⬆️ **Back to Top Button** – ปุ่มกลับขึ้นด้านบน ช่วยให้ผู้ใช้สามารถเลื่อนกลับไปจุดเริ่มต้นได้อย่างรวดเร็ว
